# StockWiKi.kr

전업투자자를 위한 금융 용어 사전과 실전 계산기.

- **288개 금융 용어** — 밸류에이션·선물옵션·거시경제·회계·퀀트·행동경제 16개 카테고리
- **28개 계산기** — PER·DCF·WACC·Black-Scholes·Greeks·Kelly·VaR·Sharpe 등
- **지식 그래프형 상세 페이지** — 용어 간 관계와 시장 영향을 연결

## 기술 스택

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- 용어 SSG 페이지로 SEO 최적화
- JSON-LD 구조화 데이터

## 로컬 실행

```bash
npm install
npm run dev
```

http://localhost:3000
